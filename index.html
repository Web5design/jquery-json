jQuery JSON plugin

JSON plugin for jQuery, provides simple ways to convert to JSON and back again.

This plugin makes it simple to convert to and from JSON:

var thing = { plugin: 'jquery-json', version: 1.3 };

var encoded = $.toJSON(thing);               // '{ "plugin": "jquery-json", "version": 1.3 }'
var compressed = $.toJSON(thing, true);     // '{"plugin":"jquery-json","version":1.3}'

var name = $.evalJSON(encoded).plugin;      // "jquery-json"
var version = $.evalJSON(encoded).version;  // 1.3

This plugin extends the jQuery object with four new methods:

  • toJSON: Serializes a javascript array, object, number, String or Date into JSON.
  • compactJSON: Serializes as toJSON does, but takes out spaces, making the result shorter.
  • evalJSON: Converts from JSON to Javascript, quickly.
  • secureEvalJSON: Converts from JSON to Javascript in a secure way.
